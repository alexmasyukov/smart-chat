# Правила проекта

## Голосовое оповещение о завершении работы (ОБЯЗАТЕЛЬНО)

По завершении любой значимой задачи, ПОСЛЕ письменного итога в чате, озвучивать
**очень краткий** итог голосом через Bash:

```bash
VOICE=F1 SPEED=1.2 STEPS=20 /Users/alex/my-pro/smart-chat/voice-supertonic/say.sh "<краткий итог>"
```

Это НЕ заменяет письменный итог — полный результат всё равно пишется в чат.
Голос — короткий звуковой сигнал «готово», чтобы пользователь со свёрнутым
терминалом (работает в нескольких задачах, + «светофор»-приложение) понял, что
можно переключиться. Фраза короткая: «сделал то-то, готово».

Начинать со слова-сигнала завершения (можно варьировать: «Готово», «Сделано»,
«Закончил», «Есть», «Собрал» и т.п.) — главное, чтобы оно сигнализировало, что
процесс окончен / итог подведён.

Правила текста для озвучки:
- **Ударения `+` — только на спорных словах** (омографы, редкие слова, где
  автоматика Supertonic ошибается). На обычных словах не ставить — авто-расстановка
  справляется. `+` перед ударной гласной, сервер конвертирует в акут U+0301.
- `ё` не поддерживается набором символов Supertonic — писать `е`.
- Слово «братик» — только для текста в чате, в озвучку НЕ вставлять.
- Одно короткое предложение.

**Если есть вопрос к пользователю:** сначала озвучить результат (say.sh
«сделал то-то»), и только ПОТОМ задавать вопрос в тексте. Порядок: голос-итог →
вопрос.

Требует поднятый сервер Supertonic на порту 8126. Если не отвечает — поднять:
`cd /Users/alex/my-pro/smart-chat/voice-supertonic && nohup python server.py > out/server.log 2>&1 &`

Механизм — просто вызов `say.sh` через Bash (не хук, не MCP: они не знают текст
с ударениями). Подробности голосового стека — в папках `voice-supertonic/`
(быстрый русский, дефолт) и `voice-confucius/` (клон голоса, но глухой звук).
