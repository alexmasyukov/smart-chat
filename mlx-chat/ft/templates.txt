# Шаблоны со слотами для joint-модели (интент + извлечение параметров).
#
# Зачем отдельно от phrases.txt: там банк фраз БЕЗ параметров, он кормит и
# 350M, и ruBERT — его формат трогать нельзя. Здесь — фразы С параметрами,
# размеченные автоматически: генератор сам подставляет значения в {slot} и
# потому знает точные границы, руками размечать ничего не надо.
#
# Формат:
#   "# <intent>"        — секция шаблонов для интента
#   "# slot:<name>"     — банк значений слота
#   {branch}, {target}  — подстановка значения слота
#   {NNNN}              — случайный номер тикета (генерится, не берётся из списка!)
#
# Почему {NNNN} случайный: если в обучении будут только «ард 1120» и «ард 342»,
# модель выучит эти числа как слова и на «ард 9999» промахнётся. Со случайными
# номерами она учит правильное — «после ветке/ард в этой позиции стоит branch».
#
# Редактируй смело — потом .venv/bin/python ft/train_joint.py переобучит.
